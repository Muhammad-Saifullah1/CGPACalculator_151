# CGPA Calculator

A simple Python program to calculate Semester GPA and Overall CGPA based on subject GPA and credit hours. It also shows the final grade according to CGPA.

Features
1. Calculate GPA for multiple subjects
2. Supports previous semester CGPA
3.Calculates overall CGPA
4. Displays final grade (A to F)

How to Run<br>
</> Bash<br>
    python cgpa_calculator.py
